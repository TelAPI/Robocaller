RoboCaller

This is a sample application, written for Django, that uses TelApi to process a list of phone numbers, calling them one-at-a-time and play an audio file or text-to-speach message.

The list of phone numbers is uploaded as a CSV (comma separated) file, using the following format: 
			+18005551212,+18005552424
			+18005551212,+18005552525
			+18005551313,+18005552626

The first column is the phone number that will be reported to the dialed phone's caller ID. The second column is the phone number to be dialed. You can change the reported number with each line, as often as you need to.

You can specify a single audio file (that must be reachable via URL) or a text-to-speech message. If you want to play different messages for each caller, you can specify a different audio file in the third column of each line of the CSV file.
